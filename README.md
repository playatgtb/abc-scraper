# abc-scraper

## Setup and usage

```
git clone git@github.com:playatgtb/abc-scraper.git

# build
npm i

# run headless
npx playwright test

# run interactive UI
npx playwright test --ui

# run the weekly report and notifier
npm run weekly

# see package.json scripts for other more granular options
```
